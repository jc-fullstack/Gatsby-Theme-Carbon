## ResourceCard

The `<ResourceCard>` component should generally be used inside of a `<Row>` and `<Column>` with a `className="resource-card-group"` placed on the container row. This allows the correct border placement between a group of cards.

<Row className="resource-card-group">
<Column colMd={4} colLg={4} noGutterSm>
    <ResourceCard
      subTitle="Design Kit"
      title="Explore & Create"
      aspectRatio="2:1"
      actionIcon="arrowRight"
      href="https://github.com/IBM/carbon-elements/blob/master/.github/CONTRIBUTING.md"

### PropTypes

| property    | propType | required | default  | description                                                                             |
| ----------- | -------- | -------- | -------- | --------------------------------------------------------------------------------------- |
| children    | node     |          |          | Use 32x32 image as child, will display in bottom left of card                           |
| href        | string   |          |          | Set url for card                                                                        |
| subTitle    | string   |          |          | Smaller title                                                                           |
| title       | string   |          |          | Large title                                                                             |
| actionIcon  | string   |          | `launch` | Action icon, default is launch, options are `Launch`, `ArrowRight`, `Download`, `Error` |
| aspectRatio | string   |          | 2:1      | Set card aspect ratio, default is 2:1, options are 1:1, 16:9, 4:3                       |
| dark        | bool     |          | false    | Set true for dark card                                                                  |
| disabled    | bool     |          | false    | Set for disabled card                                                                   |
| className   | string   |          |          | Add custom class name                                                                   |
